# FlowPilot AI 🚀

Multi-Agent Productivity Assistant built using FastAPI and deployed on Google Cloud Run.

## Features
- Task Management Agent
- Calendar Scheduling Agent
- Notes Management Agent
- Workflow Orchestration Engine

## API Endpoints
- POST /chat
- GET /workflows
- GET /tasks
- GET /events
- GET /notes

## Deployment
- Google Cloud Run
- Dockerized FastAPI app

## Demo Use Case
"Schedule a meeting and create a task and save a note"

System automatically orchestrates multiple agents and persists results.
